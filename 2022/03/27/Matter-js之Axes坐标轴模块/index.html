Matter.js之Axes坐标轴模块

本文最后更新于：1 年前

Matter.js提供了一个操作坐标轴的一个模块——Matter.Axes，包含了创建和操作坐标轴的设置的一系列方法。

方法

fromVertices

Matter.Axes.fromVertices(vertices) → Axes

创建一个新的坐标轴集合，从给定的顶点。

参数

名称 类型 默认值 说明
vertices Vertices 顶点

返回值

返回一个点数组，其中每个点是一个向量。

名称 类型 说明
Axes Axes

rotate

Matter.Axes.rotate(axes, angle)

旋转一组坐标轴，通过给定的角度。

参数

名称 类型 默认值 说明
axes Axes 坐标轴
angle number 旋转角度，以弧度为单位
前端 matter.js Axes
