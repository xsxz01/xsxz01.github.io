三、break标记语法

在很多常规代码中，你可能见过使用了一个循环，仅仅是为了获取一个标记，这就使得代码变得很复杂。Rust专门针对这个问题退出了一个语言功能，标记也可以包含一个表达式值，就像循环一样，让多语句块可以提前返回值。
